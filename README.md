# Visual Audio with three.js

- State : **Prototype**
- Version : 1.01
- Support : PC, Mobile <sub>kinda<sub>(strugle a bit)</sub> </sub> 
- Main inspiration : StarGlitcher Community (Visuals & Playlist) \
*Can be resumed to simple untextures shapes that resonate, change color, move with the rythm of the music*

### Description
A visual that sync well with synth and motivating music.
Made from scratch in one night using mainly three.js. \
*Helped with Gemini chatbot to fix unrelevant issues and make the audio player feature, as only the visual was the key product.* \
Most songs given by default are available on youtube (Links will be added later on). \
**Futher upgrades might be coming !**

## Features

- Basic audio player [Volume, Start/Pause, Select]
- Temporary music file depot
- Pad around controls
- RGB gradient

### Content
- 1 Theme (Cube surge)
- 1 Audio wave visual (Symetrical Circle + Bass sides)
- 4 Songs
